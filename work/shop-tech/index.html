SYSTEM // ADAPTIVE ANC

A feedforward loop that reads the room 48,000×/sec

Dual reference microphones sample ambient pressure and feed an on-chip DSP that re-computes the anti-noise curve every 20 microseconds. The result is a −48 dB attenuation floor that holds steady as you move between a cabin, a street, and an open office — no pumping, no hiss.
